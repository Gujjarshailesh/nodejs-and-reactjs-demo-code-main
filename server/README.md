# Technource API

Clone down this repository. You will need `node` and `npm` installed globally on your machine.

## Requirements

- Node with minimum version **v16.15.0**

- Npm with minimum version **8.5.5**

## Installation:

Install all packages:
​

    npm  install

Migrate database

    npm run migrate

To Run Seeder:
​

    npx sequelize db:seed:all

To start the server:
​
npm run dev

To Visit App:​

    http://localhost:3001/
